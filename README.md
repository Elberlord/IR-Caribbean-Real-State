# I&R Caribbean Real Estate

Static website ready for GitHub Pages.

## Files

- `index.html` → main page
- `styles.css` → all styling
- `script.js` → contact data and small interactions

## Before publishing

Open `script.js` and replace these placeholder values:

- `whatsappNumber`
- `whatsappDisplay`
- `email`
- `emailSubject`
- `emailBody`

## Publish on GitHub Pages

1. Create a repository on GitHub.
2. Upload all files from this folder to the repository root.
3. Go to **Settings** → **Pages**.
4. In **Build and deployment**, choose:
   - Source: **Deploy from a branch**
   - Branch: **main**
   - Folder: **/root**
5. Save.
6. GitHub Pages will generate your public site URL.
